** วิธีการใช้โปรแกรม
การทำงานของโปรแกรมจะแบ่งออกเป็น 2 ส่วน
 
- ส่วนที่ 1 เป็นการเพิ่มข้อมูลลงฐานข้อมูล MySQL 
เมื่อทำการ run program จะขึ้นข้อความ Would you like to Insert or Query data? : "i" for Insert or "q" for Query ?? จากนั้นให้ทำการกรอกตัว "i" จะเข้าสู่กระบวนการเพิ่มข้อมูลลงฐานข้อมูล MySQL 

- ส่วนที่ 2 เป็นการดู insight เมื่อทำการ run program จะขึ้นข้อความ Would you like to Insert or Query data? : "i" for Insert or "q" for Query ?? จากนั้นให้ทำการกรอกตัว "q" และให้กรอกตัวเลข 1-5 โดยจะมีรายละเอียด insight ตามด้านล่าง 
1. สมาชิกแต่ละคนใช้เงินทั้งหมดไปเท่าไหร่ในการซื้ออาหารที่มีแคลอรี่ต่อชิ้นไม่เกิน xxx แคลอรี่
2. ประเภทอาหารที่สมาชิกกิน แต่ละชนิดมีราคาทั้งหมด แคลอรี่ทั้งหมด จำนวนการสั่งอาหารประเภทนี้ และแคลอรี่ต่อจำนวนสั่ง เท่าไหร่
3. ในมื้ออาหารแต่ละมื้อสมาชิกแต่ละคนกินไปทั้งหมดกี่แคลอรี่
4. สมาชิกแต่ละคนสั่งอาหารตามช่องทางต่างๆไปจำนวนเท่าไหร่
5. ในช่วงเวลาหนึ่งสมาชิกแต่ละคนสั่งเมนูต่างๆไปจำนวนกี่ครั้ง

* หากต้องการดู insight ต่อไปเรื่อยๆหรือยกเลิกการทำงาน program เมื่อมีคำถาม Do you want to ask again ? please insert "Y" or Any to quit... ให้ทำการกรอก "y" เพื่อให้ program ทำงานต่อไป หรือหากต้องการยกเลิกการทำงาน program ให้กรอกตัวอักษรอื่นๆ
